Boiler Plate of HTML

Boilerplate code is a standardized, reusable block of code that's used as a starting template for a new document. In HTML, the boilerplate provides the basic foundational structure that every web page needs to function correctly in a browser.

You can typically generate it in a code editor like VS Code by typing "!" and pressing "Enter".

What comes when you press "!" and then hit "Enter"

<!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
        </head>
        <body>

        </body>
        </html>

BreakDown of Boiler Plate

  • <!DOCTYPE html>: This is the very first thing in your document. It declares that the page is an HTML5 document, which tells the browser how to read and render the code.
  • <html lang="en"></html>: This is the root element that wraps all the content on the page. The lang="en" attribute specifies that the page's primary language is English, which helps with accessibility and search engines.
  • <head>: This section contains meta-information about the web page that is not displayed directly on the page itself. It includes things like the title, character set, and links to stylesheets.
  • <meta charset="UTF-8">: This specifies the character encoding for the document. UTF-8 is a universal character set that includes almost any character from any human language, including emojis, preventing issues with displaying text.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: This is crucial for responsive design. It tells the browser to set the width of the page to the screen-width of the device and set the initial zoom level to 1. This makes your website look good on all devices, from desktops to mobile phones.
  • <title>Document</title>: This sets the title of the web page, which appears in the browser tab and in search engine results.
  • <body>: This is the main content of the page. It contains all the visible elements of the page, such as headings, paragraphs, images, links, and more.

File and Path in HTML

File is a single unit of data that contains information about a specific thing. For example, a file might be a photo, a document, or a program.

Path is a way to navigate to a specific file or folder within a computer's file system. It is a sequence of folders and subfolders that lead to the desired file or folder.

There are two main types of file paths: Absolute and Relative.

Absolute Path

Absolute paths are the most specific and direct way to navigate to a file or folder. They specify the exact location of the file or folder on the computer's file system.

Analogy: It's like giving someone a full mailing address, including the country, city, street, and house number.

Example: C:\Users\username\Documents\project\index.html

Relative Path

Relative paths are the least specific and indirect way to navigate to a file or folder. They are relative to the current location of the file or folder.

Analogy: It's like giving someone a partial mailing address, including the city, street, and house number, but leaving out the country and the username.

üè∑Ô∏èDiv Tag

The <div> tag in HTML, which stands for division, is used to group elements on a web page. It is a container that can hold other HTML elements inside of it.

Example:

<div>This is a div element</div>

This is a div element

Why we use Div

The primary purpose of a <div> is to create a block-level container around a section of your page. This is useful for:

  1. Styling with CSS: You can apply styles (like backgrounds, borders, or specific layouts) to an entire group of elements at once.
  2. Structuring a Layout:It's fundamental for creating page layouts, such as headers, footers, sidebars, and content areas.
  3. Manipulating with JavaScript: It makes it easy to select and control an entire section of your page with JavaScript.

Example:

<div class="product-card">
        <img src="images/shoe.jpg" alt="A running shoe">
        <h2>Cool Running Shoe</h2>
        <p>This is the best shoe for running, buy it now!</p>
        </div>

ü•∑Class

The class attribute in HTML is used to assign a name or a label to one or more elements. It's one of the most common ways to group elements together so you can apply the same styles or behaviors to them.Think of it like a "category" or a "group name." You can give the same class to many elements.

Use of Class

Many-to-One: Multiple elements on a page can have the same class name.

One-to-Many: A single element can have multiple classes, separated by spaces.

Styling: Its primary use is to allow CSS to select and style specific groups of elements.

JavaScript: It's also used by JavaScript to find and manipulate groups of elements.

How It's Used

Let's say you have several notes on your page, but you want some of them to look more important. You can assign them a class.

        <div class="note">
        <p>This is a standard note.</p>
        </div>

        <div class="note important">
        <p>This is an important note.</p>
        </div>

        <div class="note">
        <p>This is another standard note.</p>
        </div>

In your CSS, you could then style them like this:

  1. Use .note to style all the notes (e.g., give them all a gray border).

  2. Use .important to add extra styling only to the important notes (e.g., make the border red and the text bold).
