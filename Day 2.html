ğŸ§Element of HTML Webpage

All HTML documents must start with a document type declaration: <!DOCTYPE html>.

The HTML document itself begins with <html> and ends with </html>.

The visible part of the HTML document is between <body> and </body>.

The HTML element is everything from the start tag to the end tag:

<tagname> goes here...</tagname>

ğŸ·ï¸Heading Tag

Defination:

The h1 to h6 tags are used to define HTML headings.

h1 defines the most important heading. h6 defines the least important heading. Note: Only use one h1 per page - this should represent the main heading/subject for the whole page. Also, do not skip heading levels - start with h1, then use h2, and so on.

ğŸ·ï¸Horizontal (hr) Tag

The <hr> tag defines a thematic break in an HTML page (e.g. a shift of topic). The <hr> element is most often displayed as a horizontal rule that is used to separate content (or define a change) in an HTML page.

ğŸ·ï¸Break (br) Tag

The <br> tag in HTML stands for "break" and is used to insert a single line break.It forces any content that follows it to start on the next line. Think of it as pressing the Enter key once in a word processor.

How It Works

The <br> tag is an empty tag, which means it doesn't have a closing tag. You simply place it where you want the line to break.

When We use it

The <br> tag is best used for content where the division of lines is important, such as:

  • Poems: To separate verses.
  • Addresses: To put the street, city, and postal code on separate lines

ğŸ·ï¸Image Tag

The <img> tag is used to embed an image in an HTML page. Images are not technically inserted into a web page; images are linked to web pages. The <img> tag creates a holding space for the referenced image.

Atributes in Image Tag

  • src Atributes:

    The src (source) attribute is required and specifies the path to the image you want to display. The path can be a URL to an image on the web or a local path to an image file on your server.

  • alt Attribute

    The alt (alternative text) attribute is also required. It provides a text description of the image. This text is crucial for two reasons:

    1. Accessibility: Screen readers for visually impaired users will read the alt text aloud.
    2. Broken Images: If the image fails to load, the alt text will be displayed in its place.

ğŸ·ï¸<a> Anchor Tag

The <a> tag in HTML, which stands for anchor, is used to create a hyperlink. This is what makes the web "clickable," allowing you to link from one page to another.

Example:

<a href="https://www.google.com">Click here to go to Google</a>
