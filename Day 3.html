üè∑Ô∏èList in HTML

In HTML, lists are used to group a set of related items. There are three main types of lists: unordered, ordered, and description lists.

Types of List

  1. Ordered List

  2. Unordered List

  3. Description List

Ordered List

An ordered list is used when the sequence of the items is important. It creates a numbered list. The list is defined with the <ol> tag, and each item is also defined with the <li> tag.</p>

Example


        <ol>
        <li>Preheat the oven.</li>
        <li>Mix the ingredients.</li>
        <li>Bake for 30 minutes.</li>
        </ol>

Output

  1. Preheat the oven.
  2. Mix the ingredients.
  3. Bake for 30 minutes.

Unordered List

An unordered list is used when the order of the items doesn't matter. It creates a bulleted list. The list is defined with the <ul> tag, and each item within it is defined with the <li> (list item) tag.

Example

Example kuch iss prakar hoga

        <ul>
        <li>Milk</li>
        <li>Bread</li>
        <li>Cheese</li>
        </ul>

Output

  • Milk
  • Bread
  • Cheese

Description List <dl>

A description list is used for grouping terms with their corresponding descriptions, like in a dictionary or glossary.

  • <dl> (description list): The container for the entire list.
  • <dt> (description term): The term to be defined.
  • <dd> (description details): The description of the term.

Example

Iske Example Ye hai 

        <dl>
        <dt>HTML</dt>
        <dd>HyperText Markup Language, the standard for creating web pages.</dd>
        <dt>CSS</dt>
        <dd>Cascading Style Sheets, used for styling the look of a web page.</dd>
        </dl>

Output

HTML
HyperText Markup Language, the standard for creating web pages.
CSS
Cascading Style Sheets, used for styling the look of a web page.

üè∑Ô∏èTable in HTML

The <table> tag in HTML is used to display data in a grid-like format of rows and columns.

Essential HTML Table Tags

Tag Description
<table> The main container for the entire table.
<tr> Creates a row within the table.
<th> Defines a header cell. Text inside a <th> is typically bold and centered by default. It should be placed inside a <tr>.
<td> Defines a standard data cell. It should also be placed inside a <tr>.

Basic Table Structure

<table>
    <tr>
        <th>First Name</th>
        <th>Last Name</th>
    </tr>
    <tr>
        <td>John</td>
        <td>Doe</td>
    </tr>
    <tr>
        <td>Jane</td>
        <td>Smith</td>
    </tr>
    </table>

Output

First Name Last Name
John Doe
Jane Smith

Advanced & Semantic Structure

For more complex and accessible tables, you can use additional tags to structure the content semantically.

<thead>: Groups the header content (<th> rows).
<tbody> Groups the main body content (<td> rows).
<tfoot> Groups the footer content of the table.

These tags help browsers and screen readers understand the table's structure, but they don't change the visual layout by default.

Spanning Columns and Rows

You can make cells span across multiple columns or rows using special attributes:

colspan Merges a cell with cells to its right.
rowspan Merges a cell with cells below it.

Example with all Feature

Hum Yaha par sabhi feature table ke ek baar me dekhenge 

<table>
    <thead>
        <tr>
        <th>Name</th>
        <th colspan="2">Contact Details</th>
        </tr>
    </thead>
    <tbody>
        <tr>
        <td rowspan="2">John Doe</td>
        <td>Email</td>
        <td>john.doe@example.com</td>
        </tr>
        <tr>
        <td>Phone</td>
        <td>555-1234</td>
        </tr>
    </tbody>
    </table>

Output

Name Contact Details
John Doe Email john.doe@example.com
Phone 555-1234
